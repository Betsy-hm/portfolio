# Portfólio Pessoal - Betsy

O projeto consiste na criação de uma landing page funcional com o objetivo de apresentar um portfólio pessoal na área de programação. O propósito central é consolidar, em um único espaço digital, informações relevantes sobre a formação, as habilidades e os projetos desenvolvidos ao longo do processo de aprendizagem.

A página foi estruturada em seções claras: apresentação pessoal, descrição profissional, habilidades técnicas e comportamentais, além de uma área dedicada à exibição de projetos e um espaço de contato. Na seção inicial, são apresentadas informações sobre o perfil profissional, destacando a transição para a área de tecnologia e a combinação de conhecimentos multidisciplinares.

A seção de projetos apresenta exemplos de aplicações desenvolvidas, incluindo uma landing page, um relógio digital e um gerador de cores aleatórias. Para cada projeto, são informados o nome, descrição, tecnologias utilizadas e link de demonstração (quando aplicável). Já a seção de contato permite a comunicação direta com o visitante, por meio de um formulário funcional, contribuindo para a interação profissional.

O projeto foi desenvolvido utilizando as linguagens HTML, CSS e JavaScript, garantindo uma estrutura semântica adequada, estilização responsiva e interatividade básica com navegação suave entre seções. Esses elementos estão diretamente relacionados aos fundamentos da programação Front-End, responsável pela interface e experiência do usuário. Além disso, o projeto considera conceitos introdutórios de Back-End, especialmente no que diz respeito à lógica de envio de dados e possibilidade de integração com serviços externos para processamento de formulários.

Todos os arquivos estão organizados de forma clara e o sistema pode ser executado diretamente em qualquer navegador moderno, sem necessidade de configurações adicionais, atendendo aos requisitos de funcionalidade e execução sem erros críticos.

Este portfólio serve como base inicial para evolução profissional, podendo ser expandido com novos projetos, integração com GitHub, melhorias visuais e futuras implementações envolvendo comunicação completa com Back-End.
